# BellyButton Privacy Policy

Privacy Policy page for the BellyButton mobile application.

## Live URL

Once deployed to GitHub Pages, the privacy policy will be available at:

```
https://<your-username>.github.io/bellybutton-privacy/
```

## Features

- Responsive design (mobile, tablet, desktop)
- Dark mode support with toggle button
- Professional styling with BellyButton brand colors
- Print-friendly styles

## Deployment

This page is designed to be hosted on GitHub Pages:

1. Push to GitHub
2. Go to repository Settings > Pages
3. Select "main" branch as source
4. Your privacy policy will be live at the URL above
